Introduction to the Web
world wide web: collection of HTML documents
HTML: hypertext markup language; main type of document on the web
hyperlinks: connect pages on the web

What type of files can be found on the web?
- plain text <
- html
- images
- videos
- music

Major Pieces
1. You, your computer & browser
2. The Internet
3. Servers

browser: program that runs on your computer to display files found on the web
internet: world's largest computer network
http: main protocol of the web
servers: computers that host the files that make the web

You, your computer, and browser make a request by HTTP through the internet. The servers respond by sending files back.

HTML: hypertext markup language
text content : what you see
markup: what it looks like
references: point to other documents
links: connect to other pages

attributes: usually contain a value
anchor: link; href is the attribute, link address is the value
img: src is the attribute, image address is the value; alt is another attribute that is normally added for broken requests and blind users

span: inline editing
- a, span, br, img, strong

div: block editing
- div, p, form
